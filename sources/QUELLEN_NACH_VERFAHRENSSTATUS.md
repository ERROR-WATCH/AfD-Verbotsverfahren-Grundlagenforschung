# Quellen nach Verfahrensstatus

| Code | Label | Definition |
|---|---|---|
| `rechtskraeftig` | rechtskraeftig | Rechtskraeftige Entscheidung oder bestandskraeftiger Status. |
| `gerichtlich_bestaetigt` | gerichtlich bestaetigt | Gerichtliche Bestaetigung, ggf. ohne Hauptsacherechtskraft. |
| `eilverfahren_bestaetigt` | im Eilverfahren bestaetigt | Vorlaeufiger Rechtsschutz bestaetigt Behoerden-/Statushandlung. |
| `hauptsache_anhängig/statusgetrennt` | Hauptsache anhängig/statusgetrennt | Hauptsacheverfahren noch nicht abgeschlossen. |
| `amtliche_einstufung` | amtliche Einstufung | BfV/LfV/BMI/BKA oder andere Behoerde weist Status oder Lage aus. |
| `stillhaltezusage` | Stillhaltezusage | Behoerde setzt Behandlung/Kommunikation vorlaeufig aus. |
| `anklage` | Anklage | Foermliche Anklage erhoben; keine Verurteilung. |
| `ermittlungen` | Ermittlungen | Ermittlungsstand; keine gerichtliche Feststellung. |
| `freispruch_entlastung` | Freispruch/Entlastung | Entlastender oder nicht belastender Verfahrensausgang sichtbar halten. |
| `presse_recherchebeleg` | Presse-/Recherchebeleg | Faktenbasierter Bericht; Primaeranker ergaenzen. |
