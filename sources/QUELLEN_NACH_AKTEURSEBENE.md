# Quellen nach Akteursebene

| Code | Label | Definition |
|---|---|---|
| `bundespartei` | Bundespartei | AfD als Bundespartei, Bundesvorstand, Bundesparteitag, Bundesprogramm, Bundeskanal. |
| `landesverband` | Landesverband | Landespartei, Landesvorstand, Landtagsfraktion oder landesbezogene Parteistruktur. |
| `fraktion` | Fraktion | Bundestags-, Landtags-, kommunale oder EP-Fraktion/Gruppe. |
| `teilorganisation` | Teilorganisation | Junge Alternative, Generation Deutschland, Fluegel und andere parteinahe/formale Strukturen. |
| `funktionaer` | Funktionaer/Mandatstraeger | Person mit Amt, Mandat, Listenplatz, Vorstandsrolle oder offizieller Sprecherrolle. |
| `mitglied` | Mitglied | Einfaches Mitglied; Zurechnung nur mit Duldung, Funktion, Kontext oder Parteireaktion. |
| `mitarbeiter` | Mitarbeiter/Zugangsperson | Beschaeftigte, persoenliche Mitarbeiter, Hausausweis-/Zugangspersonen. |
| `vorfeld` | Vorfeld/Umfeld | IB, IfS, Ein Prozent, Compact, Active Clubs, externe Kampagnenstrukturen. |
| `kontext` | Lagebild/Kontext | Allgemeine Lagezahlen oder Umfeld ohne automatische AfD-Zurechnung. |
